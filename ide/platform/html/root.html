The Symphony IDE

Comprehensive Modelling for Advanced Systems of Systems

The Symphony IDE aims to support the building of models of System of Systems (SoSs) using the COMPASS Modelling Language (CML) notation and provide analysis of global properties on these models.

Getting Started

  • The user manuals for the different features are included as PDF files in the distribution.
  • Public CML examples can be imported using the File -> Import -> COMPASS -> COMPASS examples selection.
    • Additional examples can be found at the COMPASS examples web page

Upgrading Workspaces

Due to current development pace, new versions of Symphony may not be compatible with workspaces from previous versions. This often materializes in "Editor could not be initialized" errors. Creating new workspaces usually solves these problems. In order to do so:

  1. Delete the existing workspace folder from file system (be sure to backup any projects you wish to preserve).
  2. Start the Symphony IDE and point the workspace to a new location or recreate the old one.
  3. Recreate any projects you wish to work on. Afterwards use Import... -> File System to import the relevant CML sources into your new projects.
